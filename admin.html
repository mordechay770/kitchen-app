🔐 Admin Credentials

Admin credentials are stored securely as environment variables on Netlify — not in the code.

To change username or password:
  1. Open admin-setup.html locally to generate a new hash
  2. Go to Netlify → Site → Environment variables
  3. Update ADMIN_USERNAME, ADMIN_PASS_HASH, ADMIN_SECRET
  4. Redeploy the site

Deploy in Minutes Запуск за несколько минут פריסה בדקות

No server needed. Edit config.js with your Airtable API key and table IDs, then host on GitHub Pages — completely free. Сервер не нужен. Отредактируйте config.js с ключом API Airtable и ID таблиц, затем разместите на GitHub Pages — бесплатно. אין צורך בשרת. ערוך את config.js עם מפתח ה-API ומזהי הטבלאות, ופרוס ב-GitHub Pages — בחינם לגמרי.
